## Reply templates for common PH comments

Have these ready in a notes file. Personalize each reply — don't just paste — but starting from these saves time when you're flooded with comments.

### Q: "How is this different from [Yuka / DoneGood / etc.]?"

```
Great question. Three big differences:

1. **Data depth**: We don't just rate ingredients or claims — we pull from FEC donations, EPA enforcement actions, OSHA citations, NLRB cases, SEC filings. The kind of thing investigative journalists use.

2. **Personalization**: Take the 30-sec values quiz and EVERY grade re-weights to YOUR priorities. A climate-first user and a labor-focused user will see different Top Picks for the same brand.

3. **Categories**: 9 distinct value categories vs. 1-2 dimensions in most tools.

That said — Yuka is fantastic for food/cosmetics ingredient safety; we're complementary, not competitive.
```

### Q: "How do you handle bias?"

```
Honest answer: by sourcing only public records, not interpretive judgments.

A donation to a Senate campaign is a fact (FEC filing). Calling that "good" or "bad" is your call — we give you the data, the quiz tunes the weights to YOUR values, and the grade reflects that.

For categories where neutrality is impossible (e.g., "animal testing"), we use independent certifications (PETA, Leaping Bunny) — not our own interpretation.

Every score has citations in the in-app Sources tab.
```

### Q: "Why iOS only?"

```
Solo founder bandwidth + Capacitor architecture decisions. Android target is ~1 day of work once iOS settles in App Review — full plan in our launch docs. Subscribe to our newsletter for the Android announcement.
```

### Q: "What's your revenue model?"

```
Free forever for the core: search, scan, browse, basic grades.

Pro tier ($1.99/mo, launching with App Store): personalized scoring, full Sources tab, "Updates on brands you've saved" weekly digest, ad-free.

No selling user data. No affiliate links. Public records are free; the work is in cleaning + cross-referencing them.
```

### Q: "Is this open source?"

```
Not yet — keeping the data pipeline private during early launch. The data SOURCES are all public (you can verify our grades against FEC.gov, EPA.gov, etc. yourself). Long-term I'd love to open-source the scoring methodology so it's fully auditable.
```
