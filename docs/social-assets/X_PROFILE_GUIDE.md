- Profile → **Edit profile** → tap the avatar → upload `linkedin-logo-400.png`.
- X crops to a circle — the compass mark is centered, so it survives.

## 2. Header photo (2 min)
- **Edit profile** → tap the header area → upload `x-header-1500x500.png`.
- The avatar overlaps the **bottom-left**; the hero is pulled up-and-right so nothing important is covered. No repositioning needed.

## 3. Name + handle (3 min)

| Field | Value |
|---|---|
| **Display name** | `TruNorth` (one word — never "Tru North") |
| **Handle / @username** | `@trunorthapp` → `x.com/trunorthapp` |

Set the handle under **Settings → Your account → Account information → Username**. If `@trunorthapp` is taken, fall back to `@trunorth_app`, `@gettrunorth`, or `@trunorthHQ`. Keep it consistent with the other platforms (`trunorthapp` is the FB/IG handle + your domain).

X bios cap at **160 characters**. Use:

```
Know where your money goes. We grade 12,000+ brands on politics, environment, labor & more — using only public records. No vibes. Free on iOS.
```

(147 chars.) Alternate, more punchy:

```
We grade 12,000+ brands from 200+ public records (FEC, EPA, OSHA, SEC). Every grade traces to a filing you can check. No vibes. Free on iOS 📍

(140 chars.)

## 6. Professional / verification niceties (5 min, optional)
- **Switch to a Professional account** (Settings → switch to Professional) → category **"Software Company"** or **"Brand"**. Free; unlocks the "About" module + analytics. No need for paid X Premium pre-launch, though a blue check adds reply-section reach if budget allows.
- **Pin your launch tweet** (Post 1 below) — tap the tweet → ⋯ → **Pin to your profile**.

## 9. Grow the account (ongoing)
- **Follow + reply** in your niche: indie makers (#buildinpublic), ethical-consumer / climate / consumer-advocacy accounts, journalists who cover corporate accountability. Thoughtful replies > cold follows.
- **Cadence:** Post 1 pinned now → 1 post every 2–3 days → daily during launch week (Jun 17–23) → 3–4×/week after.
- **Mix:** 50% "receipts" (a surprising real grade + the source), 25% product, 15% founder/#buildinpublic, 10% replies/engagement.
- **Format priority on X:** native video > image/screenshot > text thread > text > **link-only (worst — X suppresses outbound links, so put links in a reply, not the main post when reach matters).**
- **Cross-link:** add @trunorthapp to your personal X bio; pin the same launch tweet on both if you have a personal following.
